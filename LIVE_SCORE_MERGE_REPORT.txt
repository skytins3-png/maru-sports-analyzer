MARU SPORTS ANALYZER - LIVE SCORE MERGE REPORT

이번 합본에 추가된 내용:

1. sports/live_score_engine.py
   - 라이브스코어 입력 모델
   - 경기번호/리그/팀/현재점수/배당/핸디캡/언오버 기준점 분석
   - 사용자가 보낸 화면 기반 MLB 샘플 데이터 포함
   - 자동구매/자동결제 없음 필드 고정

2. ui/live_score_panel.py
   - 샘플 화면 적용 탭
   - 수동 1경기 입력 탭
   - CSV/구글시트 붙여넣기 탭
   - 모바일 카드형 LIVE 분석 표시

3. actions/run_live_score_collect.py
   - GitHub Actions용 라이브스코어 수집/분석 스캐폴드

4. .github/workflows/live_score_collect.yml
   - 30분 간격 자동 실행 예시
   - workflow_dispatch 수동 실행 가능

주의:
- 현재는 사진 화면을 직접 긁는 OCR 방식이 아닙니다.
- 안전한 방식인 수동 입력 / CSV / Google Sheet 입력 구조입니다.
- 실제 실시간 API는 추후 SPORTS_API_KEY 또는 ODDS_API_KEY로 연결하면 됩니다.
- 자동구매, 자동결제, 베팅 대행 기능은 없습니다.
