# MARU SPORTS ANALYZER - Clean Real Onefile

## 목적

기존 앱이 너무 조잡하고 패널이 섞여 오류가 반복되어, app.py 하나로 안정화한 실버전입니다.

## 원칙

- 외부 사이트 자동 호출 없음
- 샘플/TEST/가짜 추천 없음
- 과거자료/예정경기/감독선수자료 분리
- 자료 없으면 분석불가
- 화면은 5개 탭만 사용
- 기존 복잡한 내부 모듈 import 없음

## 적용 방법

GitHub의 app.py 전체를 이 ZIP의 app.py로 교체하세요.

추가 파일은 필요 없습니다.

## 사용 순서

1. 과거자료 수집 탭
   - 시즌 코드 2526
   - E0, D1, SP1 선택
   - 실제 과거자료 수집/저장

2. 예정경기 탭
   - 예정 경기 CSV 저장

3. 감독·선수 탭
   - 감독/결장/핵심선수 CSV 저장

4. 분석 탭
   - 추천카드 확인

## 주의

Streamlit Cloud의 cache 폴더는 재시작 시 사라질 수 있습니다.
중요 자료는 저장데이터 탭에서 CSV로 다운로드해 보관하세요.
