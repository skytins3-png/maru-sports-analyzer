# MARU SPORTS ANALYZER - 자동 시즌 탐색 원클릭 버전

## 반영 내용

- 사이트는 football-data.co.uk 한 개만 고정
- 시즌 코드 수동 입력 제거
- URL 복사/붙여넣기 제거
- 앱이 현재 날짜 기준으로 시즌 후보 자동 생성
- 2627, 2526, 2425, 2324 등을 자동 시도
- 되는 시즌/리그만 저장
- 버튼 하나로 과거자료 수집

## 적용 방법

GitHub app.py 전체를 이 ZIP의 app.py로 교체하세요.

## 사용법

③ 과거자료 자동수집 탭에서
'자동으로 되는 시즌/리그 찾아서 과거자료 저장' 버튼만 누르면 됩니다.
